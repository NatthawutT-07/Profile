นักวิเคราะห์ซัพพลายเชน & นักพัฒนาโซลูชัน Full-Stack

เพิ่มประสิทธิภาพการทำงานด้านซัพพลายเชนด้วยซอฟต์แวร์ Full-Stack และระบบอัตโนมัติที่ขับเคลื่อนด้วยผู้ช่วย AI เพื่อให้เกิดประสิทธิภาพสูงสุด

ติดต่อฉัน

เกี่ยวกับฉัน

บัณฑิตสาขาวิทยาการคอมพิวเตอร์ที่มีความสนใจในการจัดการซัพพลายเชน (SCM) ผมเป็นคนที่มีแรงผลักดันในตัวเองและสามารถเชื่อมช่องว่างระหว่างข้อมูลกับการปฏิบัติงานในซัพพลายเชน ด้วยความชื่นชอบในตัวเลขผสมผสานกับการพัฒนา Full-Stack และระบบอัตโนมัติ ผมสร้างโซลูชันที่ปรับแต่งได้เพื่อเปลี่ยนงานเอกสารที่ทำด้วยมือให้เป็นกระบวนการทำงานที่มีประสิทธิภาพและแม่นยำสูง ผมเรียนรู้เร็วและมุ่งมั่นที่จะเติบโตในสายงาน SCM ในระยะยาว โดยมีเป้าหมายที่จะใช้การวิเคราะห์ขั้นสูงและเทคโนโลยีเพื่อขับเคลื่อนความสำเร็จทางธุรกิจ

ทักษะทางเทคนิค

เทคโนโลยีหลัก

Node.js (Express), React + Vite, Prisma ORM, PostgreSQL, Python, SQL

ซัพพลายเชนและข้อมูล

เจ้าหน้าที่วิเคราะห์ซัพพลายเชน

ก.ย. 2025 - ปัจจุบัน

บริษัท ไบร์ทมายด์ รีเทล จำกัด

  • ระบบพาโนแกรม (Web POG) Full-Stack: พัฒนาระบบจัดการแผนผังชั้นวางสินค้าบนเว็บแบบ Custom ด้วย Node.js และ React สำหรับทุกสาขา สร้างหน้า Admin Dashboard สำหรับวิเคราะห์ยอดขายและสต็อก และหน้าจอสำหรับพนักงานหน้าร้านเพื่อติดตามตำแหน่งสินค้า
  • ระบบพาโนแกรม (Mobile POG) Full-Stack: พัฒนาระบบเวอร์ชันที่ปรับให้เหมาะสมกับมือถือเพื่อให้ใช้งานได้สะดวกยิ่งขึ้น
  • ระบบ AutoPO และจัดการข้อมูล: สร้างเครื่องมืออัตโนมัติเพื่อรวบรวมข้อมูลที่กระจัดกระจายจากหลายแหล่งเข้าสู่ฐานข้อมูลฐานข้อมูลกลางไว้ในแหล่งเดียว นำแนวโน้มยอดขายและข้อมูลการตัดจำหน่ายมาคำนวณเพื่อสร้างใบสั่งซื้อ (PO) แบบอัตโนมัติ
  • การวางแผนอุปสงค์ & รายงาน: วิเคราะห์ข้อมูลประสิทธิภาพระดับสาขา แบรนด์ และ SKU เพื่อติดตามแนวโน้มยอดขายและแก้ปัญหาสต็อกที่สำคัญ (สินค้าขาด/สินค้าล้น)
  • ปฏิบัติการด้านการตลาดและ IT: พัฒนาสคริปต์ Python เพื่อสร้างรหัสคูปองกว่า 50,000 รหัสอัตโนมัติสำหรับแคมเปญการตลาดขนาดใหญ่ และจัดการการทำงานของระบบ
  • ระบบอัตโนมัติในที่ทำงาน: เปลี่ยนงานเอกสารที่ซ้ำซากและงานออฟฟิศให้เป็นระบบดิจิทัลอัตโนมัติ เพื่อลดความผิดพลาดและเวลาที่ใช้ในการประมวลผล

🌐 NoraStory: แพลตฟอร์มเล่าเรื่องแบบไดนามิก (2026)

  • ภาพรวมโครงการ: ระบบสร้างหน้าเว็บกึ่งสำเร็จรูปอัตโนมัติสำหรับธุรกิจ E-commerce ที่ต้องการสร้างความประทับใจให้ลูกค้าผ่านการเล่าเรื่องคนหนึ่งสู่อีกคนหนึ่ง รองรับรูปแบบ URL ที่ไม่ซ้ำกันกว่า 800 ล้านรูปแบบ พร้อมระบบจัดการหลังบ้านที่เชื่อมต่อกับ Firebase อย่างสมบูรณ์
  • ระบบลิงก์ไดนามิก: พัฒนาระบบ "Dynamic Link" ที่สร้าง URL เฉพาะสำหรับลูกค้าเพื่อเข้าถึงหน้าเว็บส่วนตัว
  • Serverless Backend: สร้างสถาปัตยกรรมไร้เซิร์ฟเวอร์ด้วย Firebase Cloud Functions เพื่อจัดการคำสั่งซื้อ การชำระเงิน และเวิร์กโฟลว์อัตโนมัติ
  • Admin Dashboard: จัดการสิทธิ์การเข้าถึงตามบทบาทเพื่อดูแลคำสั่งซื้อ ตรวจสอบสลิปโอนเงิน และกำหนดเทมเพลตอย่างปลอดภัย
  • โดเมนที่กำหนดเอง: สร้างระบบ Custom Domain สำหรับลูกค้า (เช่น norastory.com/{customer}) โดยใช้การเข้าถึงแบบระบุ path ของลูกค้า
  • การจัดการฐานข้อมูล: ออกแบบฐานข้อมูลที่ปลอดภัยในระบบ พร้อมกฎระเบียบที่เข้มงวดเพื่อปกป้องข้อมูลลูกค้า มีระบบจัดการเทมเพลตที่อนุญาตให้แอดมินเปลี่ยนรูปแบบเว็บได้ทันทีโดยไม่ต้อง Deploy โค้ดใหม่

🛍️ เว็บแอปพลิเคชัน E-Commerce (2025)

  • ภาพรวมโครงการ: พัฒนาระบบ E-commerce แบบครบวงจรที่มีฟีเจอร์สำหรับทั้งผู้ใช้ทั่วไป (การเลือกซื้อ, ตะกร้าสินค้า) และผู้ดูแลระบบ (การจัดการสินค้าและคำสั่งซื้อ)
  • ความปลอดภัยและการยืนยันตัวตน: สร้างระบบล็อกอินที่ปลอดภัยด้วยการยืนยันตัวตนแบบ Token เพื่อปกป้องข้อมูลผู้ใช้และแยกสิทธิ์ระหว่าง Admin/User
  • สถาปัตยกรรมและโครงสร้างโค้ด: ออกแบบโครงสร้างโปรเจกต์แบบ "Feature-based" เพื่อให้โค้ดเป็นระเบียบ อ่านง่าย และง่ายต่อการอัปเดตในอนาคต
  • การจัดการ State: จัดการ Global State ด้วย Zustand เพื่อรองรับตะกร้าสินค้าและเซสชันของผู้ใช้ได้อย่างมีประสิทธิภาพโดยไม่ทำให้แอปล่าช้า
  • การออกแบบ UI/UX: สร้างดีไซน์ที่รองรับทุกขนาดหน้าจอ (Responsive Design) ด้วย Tailwind CSS ทำให้เว็บไซต์แสดงผลได้สมบูรณ์แบบทั้งบนมือถือและเดสก์ท็อป
